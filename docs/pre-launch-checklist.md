# Pre-Launch Checklist

## Resend Setup
- [ ] `RESEND_API_KEY` is configured in production.
- [ ] `CONTACT_RECEIVER_EMAIL` is configured correctly.
- [ ] Test inquiry email is received successfully.

## Environment Variables
- [ ] `NEXT_PUBLIC_SITE_URL` is set to the live domain.
- [ ] Production environment variables are present in Vercel.

## Domain Setup
- [ ] Domain is connected in Vercel.
- [ ] DNS records are verified.
- [ ] SSL is active.

## Vercel Setup
- [ ] Production deployment completes successfully.
- [ ] Build logs are clean.
- [ ] Environment variables are added to the correct environments.

## Google Search Console
- [ ] Domain property verified.
- [ ] Sitemap submitted.
- [ ] Key URLs requested for indexing.

## Google Analytics
- [ ] Analytics script is installed.
- [ ] Page views are tracked.
- [ ] Conversion events are planned for contact submissions.

## Indexing
- [ ] Robots file allows public pages.
- [ ] Sitemap includes all required routes.
- [ ] No public pages are blocked accidentally.

## SEO Validation
- [ ] Titles are unique.
- [ ] Meta descriptions are unique.
- [ ] Canonical URLs are correct.
- [ ] Open Graph tags are present.

## Schema Validation
- [ ] Organization schema is valid.
- [ ] LocalBusiness schema is valid.
- [ ] Product schema is valid.
- [ ] FAQ schema is valid.
- [ ] Breadcrumb schema is valid.

## Mobile Validation
- [ ] Contact form works on mobile.
- [ ] Product inquiry form works on mobile.
- [ ] Navigation and buttons are easy to tap.
- [ ] No horizontal overflow at common mobile widths.

## Final Content Checks
- [ ] Phone number is set in the site config if available.
- [ ] Contact email is correct.
- [ ] Certificate and product placeholders are ready for replacement.
