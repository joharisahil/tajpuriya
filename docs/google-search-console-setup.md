# Google Search Console Setup

## 1. Verify the domain
- Open Google Search Console.
- Add the domain property for the live website domain.
- Complete DNS verification with the provider used for the domain.

## 2. Submit the sitemap
- Submit the sitemap URL: `https://your-domain.com/sitemap.xml`
- Confirm the sitemap is discovered and processed without errors.

## 3. Request indexing
- Inspect key URLs:
  - `/`
  - `/about`
  - `/products`
  - `/industries`
  - `/certifications`
  - `/contact`
- Request indexing for the homepage and top product pages after launch.

## 4. Monitor coverage
- Check indexing status for all pages.
- Resolve any coverage errors or canonical warnings.
- Confirm no accidental `noindex` or blocked pages.

## 5. Monitor performance
- Review impressions, clicks, CTR and average position.
- Compare branded and non-branded queries.
- Watch product pages for search appearance improvements.

## 6. Ongoing checks
- Review sitemap processing after adding new pages.
- Inspect structured data reports for FAQ, Product and LocalBusiness markup.
- Re-submit updated URLs when major content changes are published.
